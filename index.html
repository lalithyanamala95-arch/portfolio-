Projects

Diet Planner

This project is a smart diet planning system that collects user inputs such as budget and basic dietary preferences, and generates a customized diet plan accordingly. The system focuses on providing nutritionally balanced meal recommendations while ensuring affordability, making healthy eating accessible to users with different budget constraints.

Diet Planner with Direct Orders

This project is an advanced version of the diet planner that not only recommends diet plans based on user input and budget but also enables users to directly order food through the website. It integrates APIs from food delivery platforms like Swiggy and Zomato to fetch food options within the user's budget, creating a seamless experience from diet planning to food ordering.

Fitness Partner Mobile App (Tinder-like App for Gym Users)

This mobile application is designed for gym enthusiasts to find compatible workout partners in their local area or within the same gym. The app uses a Tinder-like matching system and includes features such as in-app chat, subscription plans, and an AI-based training mode. The goal of the project is to improve workout consistency, motivation, and social engagement among fitness-focused users.
